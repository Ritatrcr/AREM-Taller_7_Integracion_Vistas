## 📚 Referencias utilizadas

1. The Open Group. *The TOGAF® Standard*. The Open Group, 2022. Disponible en: https://www.opengroup.org/togaf  
   **Uso en el taller:** Se utilizó como referencia para estructurar la arquitectura en vistas de negocio, datos, aplicaciones y tecnología, y para justificar la integración entre capas dentro del modelo de Elite Airsoft.

2. The Open Group. *The ArchiMate® Enterprise Architecture Modeling Language*. The Open Group. Disponible en: https://www.opengroup.org/archimate-forum/archimate-overview  
   **Uso en el taller:** Se tomó como referencia para representar relaciones entre dominios de negocio, información, aplicaciones e infraestructura, evitando que las vistas se analizaran como elementos aislados.

3. Brown, Simon. *The C4 Model for Visualising Software Architecture*. Disponible en: https://c4model.com/  
   **Uso en el taller:** Se utilizó como guía para explicar la arquitectura de software en niveles comprensibles, conectando el sistema de reservas de Elite Airsoft con componentes como página web, panel administrativo, base de datos, pagos y consentimientos digitales.

4. Object Management Group. *Business Process Model and Notation™ BPMN™ Version 2.0.2*. OMG, 2014. Disponible en: https://www.omg.org/spec/BPMN/2.0.2/About-BPMN  
   **Uso en el taller:** Se usó como referencia conceptual para documentar procesos de negocio, especialmente el flujo de reserva, pago, consentimiento, check-in y ejecución del evento.

5. Amazon Web Services. *AWS Well-Architected Framework*. Amazon Web Services, 2024. Disponible en: https://docs.aws.amazon.com/wellarchitected/latest/framework/welcome.html  
   **Uso en el taller:** Se utilizó como referencia para evaluar criterios de arquitectura cloud: excelencia operativa, seguridad, fiabilidad, eficiencia, optimización de costos y sostenibilidad.

6. Amazon Web Services. *AWS Well-Architected*. Amazon Web Services. Disponible en: https://aws.amazon.com/architecture/well-architected/  
   **Uso en el taller:** Se tomó como apoyo para proponer una infraestructura cloud escalable y razonable para Elite Airsoft, considerando monitoreo, disponibilidad, costos y crecimiento gradual.

7. International Organization for Standardization. *ISO/IEC 27001:2022 - Information Security Management Systems*. ISO, 2022. Disponible en: https://www.iso.org/standard/27001  
   **Uso en el taller:** Se utilizó como referencia para justificar controles de seguridad de la información, gestión de riesgos, confidencialidad, integridad y disponibilidad de los datos del cliente.

8. OWASP Foundation. *OWASP Top Ten Web Application Security Risks*. OWASP. Disponible en: https://owasp.org/www-project-top-ten/  
   **Uso en el taller:** Se tomó como referencia para identificar riesgos básicos en aplicaciones web, especialmente control de acceso, fallas criptográficas, errores de configuración, autenticación y monitoreo.

9. Ministerio de Tecnologías de la Información y las Comunicaciones de Colombia. *Modelo de Seguridad y Privacidad de la Información - MSPI*. MinTIC. Disponible en: https://gobiernodigital.mintic.gov.co/seguridadyprivacidad/704/w3-propertyvalue-49984.html  
   **Uso en el taller:** Se utilizó como referencia nacional para orientar buenas prácticas de seguridad y privacidad de la información, especialmente en el tratamiento de datos personales y gestión de activos de información.

10. Congreso de la República de Colombia. *Ley Estatutaria 1581 de 2012 - Por la cual se dictan disposiciones generales para la protección de datos personales*. Disponible en: https://sedeelectronica.sic.gov.co/transparencia/normativa/ley-estatutaria-1581-de-2012  
    **Uso en el taller:** Se utilizó para fundamentar la necesidad de proteger datos personales de clientes, jugadores, acudientes y registros asociados a reservas, pagos y consentimientos.

11. Presidencia de la República de Colombia. *Decreto 1377 de 2013 - Por el cual se reglamenta parcialmente la Ley 1581 de 2012*. Disponible en: https://www.funcionpublica.gov.co/eva/gestornormativo/norma.php?i=53646  
    **Uso en el taller:** Se usó como referencia normativa para el tratamiento de datos personales, autorizaciones, políticas de tratamiento y responsabilidades frente a la información recolectada por Elite Airsoft.

12. Superintendencia de Industria y Comercio. *Sobre la protección de datos personales*. SIC. Disponible en: https://www.sic.gov.co/content/sobre-la-protecci%C3%B3n-de-datos-personales  
    **Uso en el taller:** Se consultó como fuente institucional sobre el régimen colombiano de protección de datos personales y las obligaciones de los responsables del tratamiento de información.

13. diagrams.net. *draw.io / diagrams.net*. Disponible en: https://www.diagrams.net/  
    **Uso en el taller:** Herramienta utilizada para construir el tablero visual integrado de arquitectura empresarial de Elite Airsoft.

14. OpenAI. *ChatGPT*. Fuente asistida por IA, mayo de 2026.

- No se utilizó Wikipedia como fuente principal.
- Las referencias se relacionan directamente con las vistas trabajadas en el taller:
  - **Negocio:** BPMN y TOGAF.
  - **Información:** Ley 1581 de 2012, Decreto 1377 de 2013 y MSPI.
  - **Aplicaciones:** C4 Model y OWASP.
  - **Infraestructura:** AWS Well-Architected.
  - **Seguridad:** ISO/IEC 27001, OWASP, MSPI y normativa colombiana.
- Se incluyó la fuente asistida por IA de forma transparente, como apoyo de redacción y estructuración, no como fuente normativa principal.

## 📎 Relación de las fuentes con Elite Airsoft

Las fuentes seleccionadas permiten justificar que la solución propuesta para Elite Airsoft no es únicamente una página web de reservas, sino una arquitectura empresarial integrada. TOGAF y ArchiMate ayudan a organizar las vistas de negocio, información, aplicaciones e infraestructura; BPMN permite representar el flujo operativo de reservas; C4 Model facilita explicar los componentes de software; AWS Well-Architected orienta la infraestructura cloud; y las fuentes de seguridad y normativa permiten sustentar la protección de datos personales, los consentimientos digitales, los roles de acceso y la trazabilidad operativa.

En el caso de Elite Airsoft, estas referencias son importantes porque el negocio maneja información de clientes, pagos, reservas, posibles acudientes, consentimientos y registros operativos. Por lo tanto, la arquitectura debe ser entendible para la gerencia, útil para la operación y suficientemente sólida desde el punto de vista técnico y de seguridad.
